Write a program that keeps reading integers until a negative number is entered. After a negative number is entered, the program will print the sum of the maximum two integers entered 

import java.util.Scanner;

public class App {
    public static void main(String [] args) {
        Scanner input = new Scanner(System.in);

        // Initialize two variables to store the maximum numbers
        int max1 = Integer.MIN_VALUE;
        int max2 = Integer.MIN_VALUE;

        // Initialize a variable to store the user's input
        int num;

        // Prompt the user to enter integers
        System.out.println("Enter integers (negative number to stop):");
        num = input.nextInt();

        // Keep looping until the user enters a negative number
        while (num >= 0) {
            // Check if the current input is larger than max1
            if (num > max1) {
                // If so, set max2 to the previous value of max1
                max2 = max1;
                // And set max1 to the current input
                max1 = num;
            } else if (num > max2) {
                // If the current input is not larger than max1 but is larger than max2,
                // set max2 to the current input
                max2 = num;
            }

            // Read the next input
            num = input.nextInt();
        }

        // Calculate and print the sum of the two maximum numbers
        System.out.println("Sum of maximum two integers: " + (max1 + max2));
    }
}

Write a program that keeps reading integers until a negative number is entered. After a negative number is entered, the program will print the product of the minimum two integers entered 

import java.util.Scanner;

public class App {
    public static void main(String [] args) {
        Scanner input = new Scanner(System.in);

        // Initialize two variables to store the minimum numbers
        int min1 = Integer.MAX_VALUE;
        int min2 = Integer.MAX_VALUE;

        // Initialize a variable to store the user's input
        int num;

        // Initialize a counter to keep track of the number of inputs
        int count = 0;

        // Prompt the user to enter integers
        System.out.println("Enter integers (negative number to stop):");
        num = input.nextInt();

        // Keep looping until the user enters a negative number
        while (num >= 0) {
            count++; // increment the counter for every input

            // Check if the current input is smaller than min1
            if (num < min1) {
                // If so, set min2 to the previous value of min1
                min2 = min1;
                // And set min1 to the current input
                min1 = num;
            } else if (num < min2) {
                // If the current input is not smaller than min1 but is smaller than min2,
                // set min2 to the current input
                min2 = num;
            }

            // Read the next input
            num = input.nextInt();
        }

        // Check if the user has entered at least two numbers
        if (count < 2) {
            // If not, print an error message
            System.out.println("Please enter at least two numbers.");
        } else {
            // If the user has entered at least two numbers, calculate and print the product
            System.out.println("Product of minimum two integers: " + min1 * min2);
        }
    }
}
