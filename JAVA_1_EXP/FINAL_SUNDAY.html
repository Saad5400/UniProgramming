Write a program that keeps reading integers until a negative number is entered. After a negative number is entered, the program will print the sum of the maximum two integers entered 

import java.util.Scanner;

public class App {
    public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    int num, max1 = 0, max2 = 0, sum;

    System.out.println("Enter integers (negative to stop):");

    num = input.nextInt();
    while (num >= 0) {
        if (num > max1) {
        max2 = max1;
        max1 = num;
        } else if (num > max2) {
        max2 = num;
        }

        num = input.nextInt();
    }

    sum = max1 + max2;
    System.out.println("The sum of the maximum two integers is: " + sum);
    }
}

Write a program that keeps reading integers until a negative number is entered. After a negative number is entered, the program will print the product of the minimum two integers entered 

import java.util.Scanner;

public class App {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int min1 = Integer.MAX_VALUE, min2 = Integer.MAX_VALUE;

        while (true) {
            int num = sc.nextInt();
            if (num < 0) {
                break;
            }

            if (num <= min1) {
                min2 = min1;
                min1 = num;
            } else if (num < min2) {
                min2 = num;
            }
        }

        System.out.println(min1 * min2);
    }
}
